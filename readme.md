# Reverse Image Search (Public Info)

A secure web application for performing reverse image search
to find **publicly available information** related to an image.

This project does NOT identify individuals and does NOT use
face recognition technology.

---

## Features
- Image upload
- Reverse image search via Google Lens (SerpAPI)
- Displays public web sources only
- No image storage
- Privacy-safe by design
